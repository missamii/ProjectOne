The checkboxes represent lightbulbs. You'll start off with all of them turned on. The goal is to get them all to turn off. They player will be alerted they've won. Once all the lights are out.

There are five different levels to start off from or you can simply start with all the checkboxes on. But beware, no matter how you start it will hav it's own challenge. Good luck!
